# 요구사항
-  2D 혹은 3D 기반의 레이싱게임
- 총 3개의 스테이지를 구현한다.
- 난이도에 맞게 방해물의 수가 늘어나고 방해 오브젝트들을 다양하게 배치하여 구현
- 스테이지 별 배경은 3개(사막, 산악, 도심)
- 플레이어의 차량을 업그레이드하는 상점 시스템
- 레이싱 타임을 관리하는 타임 게이지 시스템
- 각 시스템에 따른 게임오버
- 타임 기반의 점수산정 방식 스코어 획득까지 구현
- 경쟁 레이서 자동차와 방해 NPC 자동차의 등장패턴, 이동패턴 등 게임 레벨디자인과 AI (AI 아니어도됨) (애니메이션사용?)
- 5종 이상의 아이템
- 랭킹(내림차순정렬) 시스템 구현 (1 to 5)
- 추가 기능

# 게임 기획서
- 2D 및 3D 기반의 레이싱 게임 (카메라 시점은 자유)
- 자동차 레이싱을 통하여 최종 우승을 목표로 하는 게임
- 사막(1스테이지), 산악(2스테이지), 도심(3스테이지)
- 플레이어 : 노후되어 성능이 떨어진 고물 자동차로 시작하여, 부품 구입을 통해 업그레이드가 진행되는 자동차
- 적 NPC : 상금을 쟁취하려는 경쟁 레이서, 괴기하고 성능이 뛰어난 레이싱 자동차
- UI는 획득 아이템, 타임 게이지, 상점에서 구입한 부품 등등을 표현
- 실행 시 풀 스크린 모드로 실행, 풀 스크린 상태에서 화면 해상도는 제한 없음
- 게임에서 표현해야 하는 도움말, 게임시작, 랭킹 등등 게임내에서 보여줄 수 있도록 구현.
- 게임이 시작되면 출반선 상에 2대의 자동차가 출반선에 배치
- 카운트다운과 함께 경주가 시작하며, 결승점 도달 시 순위 출력 및 획득 상금을 출력
- 전면, 측면 충돌을 피하고 NPC 차량들을 플레이어 차량에 후면 충돌을 유도
- 상금을 이용하여 상점에서 아이템 구입 및 업그레이드
- 플레이어 차량 1대, 경쟁 NPC 차량 1대
- 정방향 및 역 방향으로 주행 중인 비 경주 NPC 차량
- 좌우 횡 방향으로 진행 중인 비 경주 NPC 차량

# 맵
- 사막지역(1스테이지) : 주행가능한 모래 지역, 주행이 어려운(속도저하) 높은 모래 지역, 웅덩이 지역, 레이싱에 방해가 되는 오브젝트(나무, 바위, 방해 NPC 자동차 등등 표현)
- 산악지역(2스테이지) : 주행가능한 산악 지역, 속도를 저하시키는 잡초 지역, 웅덩이 지역, 레이싱에 방해하는 오브젝트(나무, 바위, 방해 NPC 자동차 등등 표현)
- 도심지역(3스테이지) : 주행가능한 도로 지역, 속도를 저하시키는 도로 옆 인도 지역, 레이싱에 방해하는 오브젝트(씽크홀, 건물, 방해 NPC 자동차 등등 표현)
- 자연스러운 스크롤로 표현

# 게임 요소
- 플레이어의 기본 동작 : 상, 하는 속도 증가 및 감소, 좌, 우는 자동차의 방향 회전
- 경쟁 NPC 차량 : 특정 AI에 의해 성능이 뛰어난 자동차 움직임과 기능을 부여하여 구현
- 비 경쟁 NPC 차량 : 정해진 방향으로 일정하게 움직임
- 각 레이싱 코스 특정 위치에서 렌덤으로 아이템 등장

# 아이템
- 아이템 1 1,000,000 만원 획득
- 아이템 2 5,000,000 만원 획득
- 아이템 3 10,000,000 만원 획득
- 아이템 4 차량 속도 순간 소폭 증가
- 아이템 5 차량 속도 순간 대폭 증가
- 아이템 6 레이싱 중에 상점 시스템에서 부품 구입 가능

# 부품
- 바퀴 부품
- 경주 도로가 아닌 옆에 도로에서도 정상 속도를 올릴 수 있는 지역 특화 바퀴의 기능
	- 사막 전용 바퀴 
	- 산악 전용 바퀴
	- 도심 전용 바퀴
- 엔진 부품
	- 6기통 엔진
		- 3000CC 파워를 가진 엔진 (배기량에 비례하여 속도가 향상됨)
	- 8기통 엔진
		- 5000CC 파워를 가진 엔진 (배기량에 비례하여 속도가 향상됨)
- 기타 부품
- 기타 부속품 게임의 재미 요소를 위해 자유롭게 추가

# 규칙
- 스테이지는 3개 레이싱 코스를 구현 해야 함 (한 스테이지 당 최소 1분 이상)
- 결승점에 1위로 도착해야 상금을 획득할 수 있고, 다음 스테이지로 진출할 수 있다.
- 경주에서 1위로 도착하지 못한 경우는 다시 현 스테이지에서 경주를 재시작할 수 있도록 구현한다.
- 화면상에 미니맵을 표현하고 플레이어와 적 NPC 플레이어의 위치를 표시하도록 구현한다
- 경주 중인 자동차와 충돌시 위치에 따라 패널티를 적용한다.
	- 전면 충돌 시 : 대폭의 속도 감소
	- 양쪽 측면 충돌 시 : 소폭의 속도 감소
	- 후면 충돌 시 : 충돌 당하는 차량은 일시적으로 속도 증가
- 도로 곳곳에서 등장하는 아이템
- 게임 플레이 시 배경음과 자동차 충돌에 대한 효과음을 구현
- 게임 플레이 시 충돌에 대한 특수효과 이펙트을 구현
- 각 레이싱 결과를 종합해서 3스테이지 게임 종료 후 랭킹 시스템에 점수를 기록하며, 현재까지의 높은 점수를 달성했을 때 1위 ~ 5위 랭킹을 정하도록 랭킹 시스템을 구현. 
- 게임 시작 전에도 랭킹 정보를 확인할 수 있도록 구현.
- 랭킹 시스템에 기록하는 점수는 레이싱 타임을 이용하여 점수로 환산할 수 있도록 점수산정 방식을 자유롭게 정해서 랭킹을 기록할 수 있도록 구현.

# 게임 플레이
- 플레이어는 좌, 우 키를 이용하여 자동차 방향을 설정.
- 플레이어는 상, 하 키를 이용하여 자동차 속도를 설정.
- 우승 상금에 대한 금액 설정은 1스테이지부터 3스테이지까지 우승 상금을 상향(예를들며, 1스테이지 500만원, 2스테이지 1000만원, 3스테이지 희망의 도시 이동 티켓 획득)으로 자유롭게 설정하여 구현한다.

# 치트키
